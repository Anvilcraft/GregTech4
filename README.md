## Contributing & Build guide
#### Setting up enviroment
```
1. git clone https://github.com/Nukepowered/GregTech4.git
2. gradlew setupDecompWorkspace
3. gradlew eclipse/idea
```
#### Building jar file

```
gradlew build
```
